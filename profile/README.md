## 🎮 About the Server

Skyverse RP runs on a fully custom framework built by Skyverse Studio — AES-encrypted event bus, 34 React NUI interfaces, 969+ inventory items, a Discord-gated priority queue, and a Cloudflare CDN for assets. Every resource is TypeScript-first with strict typing via `@citizenfx/client` and `@citizenfx/server`.

Built for serious roleplay. Designed for performance. Crafted for community.
