## ✨ About

**Skyverse RP** is a serious roleplay server running on a framework we built ourselves, line by line. No ESX. No QBCore. No forks. Every system — economy, inventory, jobs, UI — is engineered in-house so nothing about the server feels off-the-shelf.

## 🎮 The Server

Skyverse RP is built for **serious roleplay** — every resource is TypeScript-first with strict typing via `@citizenfx/client` and `@citizenfx/server`. The custom event bus is AES-encrypted end to end, all 34 NUI interfaces are hand-built in React, and assets are served through a Cloudflare CDN for consistently fast load times.

> Built for serious roleplay. Designed for performance. Crafted for community.
